# Layer Group API

Symmetry analysis and classification tables for the 80 layer groups. Shared
types such as [`moyopy.Cell`][moyopy.Cell], [`moyopy.Operations`][moyopy.Operations],
[`moyopy.UnimodularTransformation`][moyopy.UnimodularTransformation], and
[`moyopy.PointGroup`][moyopy.PointGroup] are documented on the
[API Reference](api.md) hub.

## Symmetry datasets

Run a symmetry analysis on a [`moyopy.Cell`][moyopy.Cell] treated as a layer (slab)
and inspect the result.

::: moyopy.MoyoLayerDataset
